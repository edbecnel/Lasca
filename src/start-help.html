Start Page Help

The start page (index.html) lets you choose a game variant and configure common options before launching.

Settings chosen here are saved in your browser (localStorage) and will be used when the game page loads.

1) Choose game

  • Lasca: Lasca rules on a 7×7 board (11 pieces per side).
  • Lasca 8×8: Lasca rules on an 8×8 board (12 pieces per side).
  • Damasca: Dama movement + Lasca stacking captures (8×8, 12 pieces per side), with two variants:
    • Classic: non-flying Officers (move/capture like Lasca).
    • International: flying Officers (Dama-style).
  • Dama: Dama on an 8×8 board (12 pieces per side), with two variants:
    • Dama Classic: captured pieces are removed immediately.
    • Dama International: captured pieces remain on the board until the capture sequence ends.

2) Theme

The theme dropdown selects the same theme options found on the game’s Info panel. Your choice is applied when the board loads.

3) Startup options

These are applied when the game loads:

  • Show resize icon: show the ↕️ board-height button.
  • Show board coordinates: show row/column labels on the board.
  • Threefold repetition draw: enable/disable draw by repetition.
  • Sound effects: enable subtle sounds for moves/captures.

You can also toggle Show resize icon in-game via left panel → Options → Show resize icon.

Note: move hints are currently disabled.

4) AI options

These match the AI section on the Game panel:

  • White/Black: set each side to Human / Beginner / Intermediate / Expert.
  • Speed: delay between AI moves in milliseconds.

Launch

Click Launch to open the selected game page. If a game variant isn’t available yet, Launch will be disabled.

Online multiplayer (2 players)

To play online, set Play mode to Online. You can then Create a room (Player 1), Join a room (Player 2), Spectate a public room, or Rejoin from this browser.

  • Server URL: where the online server is running (default is http://localhost:8788).
  • Create: makes a new room and starts you as Player 1.
    • Room: choose Public (spectators allowed) or Private (no public spectating).
  • Join: paste a room ID to join as Player 2.
  • Spectate: paste a room ID to spectate a public room. Private rooms require a secret watch link.
  • Rejoin room (this browser): reclaims your seat if you previously joined from this same browser (saved in localStorage).
  • Lobby: shows open rooms on the server. Click Join on a lobby row to auto-fill the join fields and launch the correct game page.

After creating a room, the game UI shows the Room ID in Info → Online. Use the copy button next to “Room ID” to copy it to your clipboard.

When you create a room, the game also shows a sticky Waiting for opponent… toast. Tap it to copy an invite link you can share.

Private-room spectating: if you created a Private room, seated players can copy a secret spectate link from Info → Online using the Ⓦ button.

While connecting, the game may show toast notifications like Connecting…, Reconnecting…, and Reconnected. You can disable toasts via Show toast notifications on the Start Page.

If you run into issues, open Info → Online and use ⓘ to view/copy a diagnostic blob you can paste into a bug report (and it will also be uploaded to the server for this room).

When joining, the Start Page will automatically open the correct variant page for that room.

Online input is blocked until both players have joined (so a single client can’t accidentally play “both colors” while waiting).

Leaving: in online games, Leave room (forfeit) ends your game immediately (counts as resign). If you simply close the tab, the server starts a disconnect grace period.

Disconnect behavior:

  • If your opponent stays connected and you don’t return before grace expires, you may lose by disconnect timeout.
  • If both players disconnect, the game stays paused and does not time out (so you can resume later).
