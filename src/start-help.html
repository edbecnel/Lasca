Start Page Help

The start page (index.html) lets you choose a game variant and configure common options before launching.

Settings chosen here are saved in your browser (localStorage) and will be used when the game page loads.

1) Choose game

  • Lasca: Lasca rules on a 7×7 board (11 pieces per side).
  • Damasca: (when available) Lasca rules on an 8×8 board (12 pieces per side).
  • Dama: Dama on an 8×8 board (12 pieces per side), with two variants:
    • Standard: captured pieces are removed immediately.
    • International: captured pieces remain on the board until the capture sequence ends.

2) Theme

The theme dropdown selects the same theme options found on the game’s Info panel. Your choice is applied when the board loads.

3) Startup options

These match the toggles on the Game panel:

  • Show move hints: show capture/landing hints on selection.
  • Animations: smooth piece movement.
  • Show board coordinates: show row/column labels on the board.
  • Threefold repetition draw: enable/disable draw by repetition.

4) AI options

These match the AI section on the Game panel:

  • White/Black: set each side to Human / Beginner / Intermediate / Expert.
  • Speed: delay between AI moves in milliseconds.

Launch

Click Launch to open the selected game page. If a game variant isn’t available yet, Launch will be disabled.
